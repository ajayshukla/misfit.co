=== WooCommerce Amazon S3 Storage ===

Contributors: Kloon
Tags: woocommerce, woo commerce, e-commerce, shop, cart, ecommerce, amazon, aws, s3, storage
Requires at least: 3.3
Tested up to: 3.3.1
Stable tag: 2.0.2

== Description ==

Store your downloadable products on the Amazon S3 service, giving clients faster download speeds and your products more security.

== Installation ==

1. Upload the folder 'woocommerce-amazon-s3-storage' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce Amazon S3 Storage' through the 'Plugins' menu in WordPress
